# Banco-Fintech

Backend de um Banco para Fintech, projeto do Curso Backend do Senai - Santo Amaro.

---

## Sobre o Projeto

Este repositório apresenta o backend de um sistema bancário desenvolvido como parte do curso de Backend do Senai. Meu objetivo com este projeto foi demonstrar competências sólidas em desenvolvimento backend, incluindo manipulação de banco de dados, implementação de operações CRUD e integração com lógica de frontend.

---

## Principais Competências Demonstradas

- **CRUD (Create, Read, Update, Delete):**  
  O sistema implementa todas as funcionalidades básicas de CRUD, essenciais para qualquer aplicação completa, permitindo o gerenciamento eficiente de dados bancários (clientes, contas, transações, etc.).

- **Backend (PHP):**  
  Todo o sistema foi desenvolvido em PHP, demonstrando compreensão dos principais conceitos backend, como rotas, controllers, conexão com banco de dados e autenticação de usuários.

- **Banco de Dados:**  
  Utilizei bancos de dados relacionais (ex: MySQL) para armazenar e gerenciar informações, incluindo criação de tabelas, relacionamentos e utilização de queries otimizadas. O projeto evidencia minha capacidade de estruturar e manipular dados do lado do servidor.

- **Frontend Básico (Integração):**  
  Apesar do foco ser backend, também desenvolvi integrações com frontend utilizando HTML, CSS e formulários, mostrando noções de como conectar a interface de usuário com o servidor para criar aplicações completas.

---

## Diferenciais
