# Banco-Fintech

Backend for a Fintech Bank, developed as a project for the Backend Course at Senai - Santo Amaro.

## Description

This project simulates the backend of a digital bank, including basic operations such as user registration, accounts, and transactions, using PHP.

## Getting Started

1. Clone the repository:
   ```bash
   git clone https://github.com/Mateus-F-Moura/Banco-Fintech.git
   ```
2. Set up your local PHP environment.

3. Open the project files and run them with your preferred web server or local development environment (e.g., XAMPP, MAMP, Laragon).

## Features

- User registration and login
- Account management
- Transaction processing

## Requirements

- PHP (version 7.x or higher recommended)
- Web server (e.g., Apache, Nginx) or local development stack
